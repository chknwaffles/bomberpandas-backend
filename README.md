## Getting Started
### Prerequisites
Frontend Repo available here: https://github.com/chknwaffles/bomberpandas

### Installing
OSX & Linux (Both frontend and backend)
```
npm install
```

## Usage
Instructions:

Player 1
WASD - Movement
Space - Bomb

Player 2
Arrow Keys - Movement
Shift - Bomb

Click play to start game!
